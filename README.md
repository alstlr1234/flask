WEB CODE_REVIWER
=====

이 리포지토리는 FLASK로 만든 웹에서 코드를 입력하면, gpt의 api를 이용해, 입력한 코드를 분석해주거나,
입력한 코드의 잘못된 부분을 수정해주는 프로그램입니다.


Installing
----------
PIP을 이용해서 설치 또는 업데이트를 합니다

FLASK Insatlling

    $ pip install -U Flask

Open Api Installing

    $ pip install openai
